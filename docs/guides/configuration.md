# Configuring Dynaconf

Dynaconf can be configured through variables suffixed with `_FOR_DYNACONF` those settings can be used to change various dynaconf behaviors.


```eval_rst
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| Environment Variable         | Type    | Usage                                                     | default               | example                                                  |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| PROJECT_ROOT_FOR_DYNACONF    | string  | Sets the project root                                     | "."                   | PROJECT_ROOT_FOR_DYNACONF="/tmp"                         |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| SETTINGS_MODULE_FOR_DYNACONF | string  | List of comma separated paths to load                     |                       | SETTINGS_MODULE_FOR_DYNACONF="config.toml,settings.yaml" |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| ENV_FOR_DYNACONF             | string  | Sets the working environment                              | "development"         | ENV_FOR_DYNACONF=production                              |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| GLOBAL_ENV_FOR_DYNACONF      | string  | Sets the prefix used for exporting parameters as env vars | "DYNACONF"            | GLOBAL_ENV_FOR_DYNACONF=MYPROGRAM                        |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| SILENT_ERRORS_FOR_DYNACONF   | Boolean | Sets if loading errors should be silenced                 | true                  | SILENT_ERRORS_FOR_DYNACONF=false                         |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| FRESH_VARS_FOR_DYNACONF      | list    | A list of vars to be re-loaded on every access            | []                    | FRESH_VARS_FOR_DYNACONF=["HOST", "PORT"]                 |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| DEBUG_LEVEL_FOR_DYNACONF     | string  | Upper case logging level                                  | NOTSET                | DEBUG_LEVEL_FOR_DYNACONF=DEBUG                           |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| DOTENV_PATH_FOR_DYNACONF     | string  | If set defines where to look for `.env` file              | None                  | DOTENV_PATH_FOR_DYNACONF="/tmp/.env")                    |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| DOTENV_OVERRIDE_FOR_DYNACONF | boolean | Sets if `.env` should override the exported envvars       | false`                | DOTENV_OVERRIDE_FOR_DYNACONF=true                        |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| AUTO_CAST_FOR_DYNACONF       | boolean | Sets if `@casting` like `@int` will be parsed             | true                  | DOTENV_OVERRIDE_FOR_DYNACONF=false                       |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| REDIS_FOR_DYNACONF_ENABLED   | boolean | Sets if redis loader is enabled                           | false                 | REDIS_FOR_DYNACONF_ENABLED=true                          |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| VAULT_FOR_DYNACONF_ENABLED   | boolean | Sets if vault server is enabled                           | false                 | VAULT_FOR_DYNACONF_ENABLED=true                          |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| REDIS_FOR_DYNACONF_HOST      | string  | Redis server address                                      | localhost             | REDIS_FOR_DYNACONF_HOST="localhost"                      |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| REDIS_FOR_DYNACONF_PORT      | integer | Redis port                                                | 6379                  | REDIS_FOR_DYNACONF_PORT=8899                             |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| REDIS_FOR_DYNACONF_DB        | Integer | Redis DB                                                  | 0                     | REDIS_FOR_DYNACONF_DB=1                                  |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| VAULT_FOR_DYNACONF_URL       | string  | Vault URL                                                 | http://localhost:8200 | VAULT_FOR_DYNACONF_URL="http://server/8200"              |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| VAULT_FOR_DYNACONF_TOKEN     | string  | vault token                                               | None                  | VAULT_FOR_DYNACONF_TOKEN=myroot                          |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| VAULT_FOR_DYNACONF_CERT      | path    | vault cert/pem file                                       | None                  | VAULT_FOR_DYNACONF_CERT="~/.ssh/key.pem"                 |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+
| VAULT_FOR_DYNACONF_VERIFY    | boolean | vault should verify                                       | None                  | VAULT_FOR_DYNACONF_VERIFY=true                           |
+------------------------------+---------+-----------------------------------------------------------+-----------------------+----------------------------------------------------------+

```
